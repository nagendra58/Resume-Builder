# Resume Builder Application

This is a React-based Resume Builder Application that allows users to create a resume directly in the browser and download it in both PDF and DOC formats.

## Features

- **Dynamic Content Editing:** Users can edit all sections of the resume directly on the page.
- **Download Options:** The resume can be downloaded as either a PDF or DOC file.
- **Customizable Sections:** Includes sections for About Me, Education, Work Experience, References, Software Skills, and Contact Information.
